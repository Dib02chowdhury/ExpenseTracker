Setup Instructions

To connect your actual Google Sheet, follow these simple steps:

  1. Open the backend code script file Code.js.
  2. Create a new Google Sheet and copy its URL ID.
  3. Click Extensions -> Apps Script in Google Sheets, paste the code, and click save.
  4. Click Deploy -> New Deployment, choose type Web App, set access to Anyone.
  5. Copy the generated Web App Exec URL and paste it in the box on the left, then click Save Configurations.
Budget Recalculation

If you modify historical records manually inside Google Sheets, you may need to force recalculate budget history to sync cumulative carry forwards.

About BudgetFlow
BudgetFlow Tracker & Planner
Version 1.2.0 (Mobile & Income Tracking Release)

BudgetFlow is a serverless, premium expense tracker and income planner application that uses Google Sheets as a database via a Google Apps Script Web App. It runs CORS-free and incorporates a full offline fallback engine.

Technical Frameworks:
Bootstrap 5 Vanilla JS Apps Script Web App Chart.js SheetJS (Excel)
